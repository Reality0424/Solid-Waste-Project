# Phase 2.1 特征优化与失效识别报告

- 芯片数: 1000, 原始指标: 28, 失效率: 9.9%
- **特征优化**: 28 原始指标 + 6 个领域交互特征(噪声裕量/时序余量/功耗/热指数等)
- 交互特征: vnm_high, vnm_low, setup_slack, dyn_power, thermal_index, leak_total
- GCN+GAT 混合架构: 自身MLP分支 + GCN分支 + GAT分支, k-NN 图(k=10), 集成 4 个
- 评估: **5 折交叉验证 OOF**(失效样本仅 99 个, 单切分不稳)

## 性能(5折 OOF)

| 模型 | 特征 | Accuracy | AUC | Recall | 角色 |
|---|---|---|---|---|---|
| 线性 Logistic | 原始28 | 0.800 | 0.845 | - | 优化前对照 |
| GCN+GAT(集成4) | 优化+图 | 0.803 | 0.826 | 0.657 | 图表示模块 |
| GBM | 原始28 | - | 0.943 | - | 参考 |
| **GBM(最终)** | **优化28+6** | **0.940** | **0.961** | 0.818 | **最终分类器** |

- **特征优化收益**(GBM): AUC 0.943 → 0.961 (+0.018); 非线性 vs 线性: +0.116
- 特征提取/识别准确率 ≥90%: ✅ 达标(GBM优化 0.940)

## 说明

- 表格型数据 + 合取/析取式失效规则上, **梯度提升树(GBM)优于神经网络/GNN**(文献普遍结论), 故最终分类器采用 GBM(优化特征); GCN+GAT 作为计划要求的图表示模块一并交付(产出芯片嵌入)。
- 三者一致表明: **单线性不够 → 需要非线性地组合多指标**, 印证项目的'多参数组合隐性故障'核心命题。
